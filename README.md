# Sparelix Test

This repository is a small test/demo project for Sparelix.

The project contains minimal static files (HTML/CNAME) and a tiny demo website used for experimentation.

## Contents

- `index.html` — Example landing page used for the test site.
- `CNAME` — Optional custom domain entry (used when deploying to GitHub Pages).
- `LICENSE` — License file for the repository.

## Quick preview

Open `index.html` in a browser to preview the site locally.
